# Changelog

All notable changes to SabTrace are documented here.

The project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Planned

- Timestamp-range filtering
- Recursive directory input
- JSON and NDJSON log-field filters
- Follow mode for active log files
- Benchmarks for large-file workloads

## [0.1.0] - 2026-07-20

### Added

- Plain-text analysis from files and standard input
- Automatic common log-level detection
- Literal, regex, level, and case-insensitive filters
- Colored terminal output
- Level statistics and JSON report export
- Script-friendly exit codes
- CMake presets and dependency-free tests
- Windows, Linux, and macOS CI
- Tagged GitHub release packaging with SHA-256 checksums
